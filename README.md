# CREATOR.AI 🚀

AI-powered viral short-form content generator built using MERN Stack + Gemini AI.

## 🌐 Live Demo

https://creator-ai-vqnf.onrender.com

---

## ✨ Features

- AI-powered script generation
- User authentication (Signup/Login)
- JWT protected routes
- User-specific dashboard
- Save generated scripts
- Delete and copy scripts
- Cinematic markdown-style AI output
- Responsive modern UI
- MongoDB database integration
- Secure API key handling with `.env`

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM
- React Markdown
- React Select

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt.js

### AI Integration
- Google Gemini API
